# Generic investigation

## Source classes
Primary documents, official pages where applicable, strong secondary, specialist references.

## Claim types
existence, date, relationship, status, quantity.

## Failure modes
Source laundering, overgeneralization, outdated URLs.

## Fact-check
Cross-check object identity, dates, and direct support for each claim.

## Sensitivity
Default low; escalate if PII or regulated data appears.

## Output emphasis
Clarity, traceability, limitations.
