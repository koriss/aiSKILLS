
# Wiki-Style Inline Citation Policy

Every important factual statement in HTML must have an inline marker:

```html
<sup><a href="#ref-1">[1]</a></sup>
```

Every marker must point to an existing bibliography item:

```html
<li id="ref-1"><a href="FULL_URL">FULL_URL</a></li>
```

Verified claims without inline citation are invalid.

Full URLs must be visible and clickable.
