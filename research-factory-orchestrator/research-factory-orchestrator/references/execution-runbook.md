2. Select output profile.
3. Compile runtime internally.
4. Create/resume state.
5. Detect tools.
6. Check permissions.
7. Build/load queue.
8. Generate search strategy.
9. Research sources.
10. Normalize sources.
11. Score sources.
12. Detect source laundering.
13. Extract evidence cards.
14. Build claims registry.
15. Draft.
16. Fact-check.
17. Locate citations.
18. Run adversarial review.
19. Audit errors.
20. Fix output.
21. Validate.
22. Run final-answer gate.
23. Deliver chat summary and artifacts.

Never stop at runtime compilation.
