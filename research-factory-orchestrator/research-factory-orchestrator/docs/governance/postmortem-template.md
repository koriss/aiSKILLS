# Blameless postmortem (template)

1. **Summary** — customer impact, duration, severity (`contracts/severity-matrix.json`).
2. **Timeline** — detection → mitigation → resolution (UTC).
3. **Root cause (5 Whys)** — immediate, contributing, systemic factors.
4. **What went well / poorly**
5. **Action items** — owner, deadline, tracking link.

Google SRE / Incident Response Commander framing.
