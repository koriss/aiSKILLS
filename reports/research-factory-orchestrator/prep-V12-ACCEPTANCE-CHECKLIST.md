
# v12 Acceptance Checklist

## Standalone HTML

- [ ] `full-report.html` opens by itself from Telegram.
- [ ] No external CSS.
- [ ] No external JS.
- [ ] No external fonts.
- [ ] No local image dependencies.
- [ ] Has viewport meta.
- [ ] Has inline CSS.
- [ ] Has wiki-style citations [1].
- [ ] Has full source URLs in references.
- [ ] Tables are mobile-safe.
- [ ] Long URLs wrap.

## Telegram

- [ ] Plain text only.
- [ ] No HTML tags.
- [ ] No Markdown links.
- [ ] No big tables.
- [ ] Each message <= 3900 chars.
- [ ] Captions <= 900 chars.
- [ ] Messages split by semantic blocks.
- [ ] Summary introduces no new facts.
- [ ] HTML report attached.
- [ ] research-package.zip attached.

## Report semantics

- [ ] Executive summary exists.
- [ ] Verdict exists.
- [ ] Key findings exist.
- [ ] Key numbers exist if relevant.
- [ ] Facts separated from analysis.
- [ ] Data separated from prose.
- [ ] Gaps/uncertainty section exists.
- [ ] Sources section exists.
- [ ] Appendices used for heavy data.
