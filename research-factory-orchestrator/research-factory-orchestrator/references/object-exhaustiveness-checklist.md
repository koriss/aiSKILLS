
# Object / Infrastructure / Project Exhaustiveness Checklist

For objects, infrastructure, products, projects, and facilities, collect:

- names and aliases;
- status;
- ownership/control;
- location descriptions as sources provide them;
- timeline;
- function;
- associated organizations;
- official mentions;
- media mentions;
- open-source references;
- disputed claims;
- former/current status;
- related entities;
- source conflicts.
