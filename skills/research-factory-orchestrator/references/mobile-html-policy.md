
# Mobile HTML Policy

The full report must be mobile-first.

Required:

```text
<meta name="viewport" content="width=device-width, initial-scale=1">
readable typography on 360px width
single-column mobile layout
responsive KPI cards
overflow-safe wide tables
touch-friendly details/summary
long URLs wrap safely
no fixed-width layout
```

Large data tables may use horizontal scroll wrappers.
Do not force all tables into cards.
