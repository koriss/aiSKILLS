# Architecture overview — v18.5.0

## Layers

1. **CLI facade** — `scripts/rfo_v18_core.py` → `runtime/cli.py` → `runtime/impl.py`.
2. **Contracts** — `contracts/*.json`, `contracts/handoffs/*`, `schemas/*`.
3. **Runtime services** — `handoff`, `trace`, `capability`, `event_history`, `output_filter`, `judge_panel`, `slo`, `merkle_anchor` (optional anchor hook).
4. **Providers** — `providers/*` delivery adapters (verify capability before real send — wiring point for production).

## Data planes

- **trace.jsonl** — hash-chained audit of validator/model decisions.
- **events.jsonl** — append-only side-effect log for replay checks.
- **outbox OUT-\*.json** + **delivery-acks** + **capability-tokens** — delivery + agency scope.

## Validation

`validate()` = structural invariants + subprocess allowlist (`validator-registry.json` subset) + `validate_skill` inventory gate.
