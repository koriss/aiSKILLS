# Telegram: краткая сводка

## Лимит

- Один Telegram-сообщение: до **4096** символов; целевой максимум **95%** → **3890** символов (запас под форматирование клиента).
- Если черновик длиннее — **сжимай по приоритету:**
  1. главный вывод и уверенность;
  2. подтверждённое / опровергнутое;
  3. гипотезы и альтернативы;
  4. неизвестное;
  5. детали (выбрасываются первыми).

## Структура (рекомендуемая)

1. **Главный вывод** (1–3 предложения).
2. **Что подтверждено** (компактно, без спама ссылками).
3. **Что не подтверждено / опровергнуто**.
4. **Лучшая гипотеза** и кратко альтернативы.
5. **Что неизвестно**.
6. **Уровень уверенности** + одна строка rationale.

## Стиль

- Аналитическая записка / разведсводка: сухо, по делу, без «мы уверены», если база слабая.
- Русский язык; данные из иностранных источников — уже переведённые формулировки.

## Чего не делать

- Поток сознания, 50 ссылок, весь HTML в сообщении.
- Скрывать неопределённость.
- Рекламный или хайповый тон.
