
# v12 Report Delivery System — Prep Kit

Generated: 2026-04-28T00:14:20Z

This is a preparation bundle, not a full OpenClaw skill release.

Goal:

```text
Move reports from placeholder HTML to a semantic, mobile-first, standalone report system:
- standalone single-file HTML;
- embedded CSS/JS in delivered full-report.html;
- semantic report model;
- report archetypes;
- reusable components;
- Telegram plain-text delivery;
- validators for HTML, Telegram, summaries, tables, citations and report structure.
```

Core rules:

```text
Full report = standalone HTML file.
Telegram = plain text briefing + attached files.
No external CSS/JS/fonts/images in final HTML.
No Telegram HTML/Markdown formatting reliance.
No large tables in Telegram.
Facts, data, evidence, analysis and uncertainty must be separated.
```
