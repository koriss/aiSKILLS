
# Telegram Message Splitting Policy

Do not split by raw character count first.

Split by semantic blocks:

```text
verdict
key_findings
facts
numbers
gaps
sources
files_notice
```

Limits:

```text
hard message limit: 3900 chars
comfortable message limit: 2600–3200 chars
ideal mobile message: 900–1800 chars
document caption limit: 900 chars
```
