
# INT Coverage Taxonomy

INT families are source/provenance coverage labels, not fake collection modes.

Directly usable as public-source families:

- OSINT;
- WEBINT;
- SOCMINT;
- FININT-public;
- GEOINT-public;
- IMINT-public;
- TECHINT-public;
- CYBINT/CTI-public;
- MEDINT-public.

Derivative-only unless direct material is supplied:

- HUMINT;
- SIGINT;
- COMINT;
- ELINT;
- FISINT/TELINT;
- MASINT;
- ACINT;
- RADINT;
- NUCINT.

ALL-INT / Multi-INT is a fusion assessment, not a collection source.
