
# PDF, Table, and Figure Policy

Claims from PDFs, tables, or figures require specific anchors.

Rules:

- PDF claims need page number or section anchor.
- Table claims need table title/number or row/column description.
- Figure/chart claims need figure number/title and what visual element supports the claim.
- If a PDF/table/figure cannot be read, the claim is not verified.
- Do not infer beyond what the table/figure supports.
- Extract quantitative claims in structured form when possible.
