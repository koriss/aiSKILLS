
# No Pipeline Claims Without Artifacts

The agent must not claim that pipeline work was done unless artifacts exist.

Forbidden without proof:

- "21 stages completed"
- "sources collected"
- "evidence cards created"
- "citation anchors located"
- "adversarial review passed"
- "final-answer gate passed"
- "files created"
- "strict-full pipeline completed"

Validator must scan final HTML and chat summary for pipeline claims and require matching artifacts in completion-proof.
