
# Social Profile Linkage Policy

A social profile can support claims only if identity is confirmed or probable.

Hard signals:

- official company site links the profile;
- profile links confirmed company/domain/project;
- confirmed publication/conference links the same profile;
- cross-linked username from confirmed account;
- profile contains exact role/company/location and is corroborated.

Soft signals:

- same name;
- same city;
- same industry;
- similar title;
- language match.

Soft signals alone cannot confirm identity.
